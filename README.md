# new

## Getting Started

### Prerequisites

- Python 3.x

### Installation

1. Clone the repository:
    ```bash
    git clone <repository-url>
    cd cardsweb-game
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

### Running the Game

Navigate to the `src` directory and run the game:

```bash
cd src
python3 main.py
